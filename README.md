TOOL DESIGNED TO GOOD PURPOSES, PENTESTS, DON'T BE A CRIMINAL !

**BeeLogger is now maintained as an educational, defensive, and research-oriented project (for pentester).**

This repository provides analysis, simulation tools to use on your system or VM only !
Demonstrations that help researchers, students, and security professionals understand :

- how input-capture techniques work 

- how persistence mechanisms are commonly used by hacker for better defense !

- how defenders can detect abnormal behaviors

**All content is strictly for authorized, legal, educational research.**
